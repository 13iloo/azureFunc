# azureFunc

Azure Functions serverless application project.

## Overview

This project demonstrates the implementation of serverless functions using Microsoft Azure Functions, providing scalable and event-driven computing solutions.

## Features

- Serverless function implementations
- HTTP triggers and bindings
- Azure service integrations
- Scalable architecture
- Cost-effective computing

## Technologies Used

- Azure Functions
- Node.js/Python/C#
- Azure services (Storage, CosmosDB, etc.)
- Serverless framework

## Functions

- HTTP-triggered functions
- Timer-triggered functions
- Queue-triggered functions
- Event-driven processing

## Getting Started

```bash
# Clone the repository
git clone https://github.com/13iloo/azureFunc.git

# Navigate to project directory
cd azureFunc

# Install Azure Functions Core Tools
npm install -g azure-functions-core-tools@4

# Install dependencies
npm install

# Run locally
func start
```

## Deployment

```bash
# Deploy to Azure
func azure functionapp publish <APP_NAME>
```
